# Quiz Name
> ### programmers / Lv. 3 <a href = "https://school.programmers.co.kr/learn/courses/30/lessons/161988"> 연속 펄스 부분 수열의 합 </a>

<br>

## 💡 approaches
>  - -1, 1 반복이므로 1, -1로 시작한 배열과 -1, 1로 시작한 배열 만들어서 계산
>  - dp

<br>

## 🔑 quiz solution

```py
def solution(sequence):
    s1 = sequence[:]
    s2 = sequence[:]
    for i in range(len(s1)):
        if i % 2 == 0:
            s1[i] *= -1
        if i % 2 == 1:
            s2[i] *= -1

    Max = s1[0]
    Sum = 0
    for i in s1:
        Sum += i
        Sum = max(Sum, 0)
        Max = max(Sum, Max)
    Sum = 0
    for i in s2:
        Sum += i
        Sum = max(Sum, 0)
    return Max
```
### Time Complexity : O(N)
