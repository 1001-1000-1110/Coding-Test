## 🔑 quiz solution

>  - chr() : 아스키코드 -> 문자로 변환한다.
>  - ord() : 문자 -> 아스키코드로 변환한다. 

```py
def solution(s, n):
    s = list(s)

    for i in range(len(s)):
        if s[i].isupper(): # 대문자일 떄 
            s[i] = chr((ord(s[i]) - ord('A') + n) % 26 + ord('A'))
        elif s[i].islower(): # 소문자일 때 
            s[i] = chr((ord(s[i]) - ord('a') + n) % 26 + ord('a'))

    return "".join(s)
```

>  - Z/z의 범위를 넘지 않기 위해 나누기 26을 한다.
